# Jobio API Client

Generated TypeScript API client for Jobio Backend.

## Installation

```bash
npm install @jobio/api-client
```

## Configuration

Add to your `.npmrc`:

```
@Jobio-Org:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=${GITHUB_TOKEN}
```

**Note:** You need to have access to the Jobio organization and set up a GitHub Personal Access Token with `read:packages` scope.
